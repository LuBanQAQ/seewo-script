# Seewo-Script (希沃品课自动答题助手)

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
![Tampermonkey](https://img.shields.io/badge/Tampermonkey-Script-green.svg)

## 📖 简介 (Introduction)

**Seewo-Script** 是一个专为希沃品课设计的 Tampermonkey (油猴) 脚本。它利用 AI 技术（支持 OpenAI 格式接口），能够自动识别题目内容，智能选择答案，并支持全自动挂机模式，助你轻松完成答题任务。

本脚本支持多种题型（单选、多选、判断、简答），并配备了现代化的 GUI 界面，支持拖拽、最小化和参数配置。

## ✨ 主要功能 (Features)

*   **🤖 AI 智能答题**：利用 AI 模型分析题目文本，精准识别并回答单选题、多选题、判断题和简答题。
*   **🚀 全自动挂机**：一键开启全自动模式，自动循环执行：`识别 -> 答题 -> 下一题`，直至完成所有题目。
*   **📝 全题型支持**：
    *   **单选题/多选题**：自动点击对应选项。
    *   **判断题**：自动识别 T/F 或 对/错。
    *   **简答题**：自动生成答案并模拟粘贴填入（支持 Draft.js 编辑器）。
*   **🖥️ 现代化 GUI**：
    *   **悬浮面板**：实时显示运行状态和操作按钮。
    *   **自由拖拽**：面板和最小化图标均支持随意拖拽。
    *   **最小化模式**：支持一键最小化，避免遮挡题目。
*   **🏁 智能交卷**：支持到达最后一题自动交卷，并自动处理二次确认弹窗。
*   **⚙️ 灵活配置**：支持自定义 API URL、Token 和模型名称，适配各种 OpenAI 格式的中转服务。

## 🛠️ 安装指南 (Installation)

1.  **安装浏览器扩展**：
    *   Chrome/Edge 用户请安装 [Tampermonkey](https://www.tampermonkey.net/)。
2.  **安装脚本**：
    *   点击 [此处安装脚本](./seewo_auto_answer.user.js) (或者手动复制 `seewo_auto_answer.user.js` 代码到 Tampermonkey 新建脚本中)。
3.  **准备 API**：
    *   你需要一个支持 OpenAI 格式的 API Key（推荐使用 `gpt-3.5-turbo` 或 `gpt-4o-mini` 模型，速度快且成本低）。

## ⚙️ 配置说明 (Configuration)

1.  进入 Seewo 答题页面，脚本会自动加载，屏幕右上角会出现悬浮面板。
2.  点击面板标题栏右侧的 **⚙️ (设置)** 按钮。
3.  填写以下信息：
    *   **API URL**: 接口地址 (例如 `https://api.openai.com/v1/chat/completions` 或第三方中转地址)。
    *   **API Token**: 你的 API Key (`sk-xxxx...`)。
    *   **Model**: 模型名称 (例如 `gpt-3.5-turbo`)。
    *   **自定义参数**: (可选) JSON 格式的额外请求参数。
4.  点击 **保存配置**。

## 🎮 使用指南 (Usage)

### 手动模式

*   **🤖 单题解答**：点击面板上的按钮，AI 将分析当前题目并自动选择或填写答案。
*   **手动导航**：使用面板上的 **← 上一题** 和 **下一题 →** 按钮进行切换。

### 全自动模式

1.  点击 **🚀 全自动开始** 按钮。
2.  脚本将自动执行：`获取题目 -> AI 思考 -> 填涂答案 -> 等待 3 秒 -> 进入下一题`。
3.  如需暂停，点击 **⏹ 停止运行** 按钮。

### 自动交卷

*   在面板底部勾选 **自动交卷** 选项。
*   当脚本运行到最后一题（检测到“下一题”按钮禁用）时，会自动点击交卷，并处理确认弹窗。

### 界面操作

*   **拖拽**：按住面板标题栏或最小化图标即可拖动位置。
*   **最小化**：点击标题栏右上角的 **➖** 按钮，面板将缩起为一个小机器人图标。
*   **恢复**：点击小机器人图标即可恢复完整面板。
